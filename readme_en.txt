
## CONFIGURATION

To start using the module, you need to configure the sender postal code and select the freight products you want to present in your front office

* Go to the module configuration
* Enter your postal code, and additionally if you want to add insurance cost and use EDI (has impact on which products that are available)
* Save, and then select the freight products you want to present in your shop
* Save again. Your shop is now ready and should show freight alternatives from Fraktguiden.
